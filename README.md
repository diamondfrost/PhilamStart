# PhilamStart Demo
A demo local web-server that will feature some add-ons to the PhilamLife Ecosystem.

## How to run
1. Install dependencies
```
pip install -r requirements.txt
```

2. Run `run.py` file
```
$ python run.py
```

3. Visit http://127.0.0.1:8000/

## Libraries Used
* Flask
* Flask-SQLAlchemy
* Flask-WTF
* WTForms
